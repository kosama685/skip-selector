# Skip Selector

A modern, animated, responsive React page for selecting a skip size.

## Features

- **Animated gradients** on cards and footer
- **Responsive** (mobile & desktop)
- **Accessible** (keyboard & ARIA support)
- **Live API data** (WeWantWaste NR32 skips)
- **Easy to extend**

## How to run

```bash
npm install
npm start
